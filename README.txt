AlexGuitars - шаблон сайта
-----------------------------

1. Все страницы находятся в папках /fr, /en, /ru
2. Все изображения — в /img (замени logo.png на свой логотип)
3. Стили — в /css/style.css
4. Язык переключается ссылками в меню
5. Добавляй новые страницы копированием index.html

Как редактировать:
- Открывай .html в любом редакторе (например, VSCode, Notepad++)
- Замени текст и фото на свои
- Потом залей сайт на GitHub Pages, Netlify или FTP-хостинг
